# Areeba Sheikh — Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features

- Dark / Light theme toggle
- Smooth scroll with Lenis  
- Scroll-reveal animations
- 3D tilt effect on cards
- Typing animation in hero
- Animated particle background
- Live project screenshots via Microlink API
- Fully responsive (mobile-friendly)

## Sections

- **Hero** — Introduction with typing effect
- **About** — Stats, story, and skills chips
- **Skills** — Web development & Office tools with progress bars
- **Projects** — 6 featured client projects with live demo links
- **Contact** — Contact info + message form
